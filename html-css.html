HTML and CSS Notes

Common HTML Elements

Heading

  • Range from <h1> to <h6>

Paragraph

  • Use <p> to start a paragraph
  • Use <br/> to start break up paragraph into a new line

Lists

  • Ordered Lists using <OL>
  • Unordered Lists using <UL>

Link

  • Use <a href="link"></a> to enclose an element to make it clickable

Image

  • Use <img src="insert link"/> to insert an image
  • Use attribute alt for the visually impaired by entering a description

Comment

  • [HTML]Use <!--Comment Here-->
  • [CSS] Use /* Comment Here */

CSS Setup

  • Use <style></style> in <head> to type css codes
  • Link stylesheet by using <link rel="stylesheet" type"text/css" href="name.css">

CSS Declarations

[Format] selector{property: value}

Colors

There are four ways to indicate a color value:

  • [Name]color: Red;
  • [Hex] color: #FFFFFF;
  • [RGB] color: rgb(123, 20, 233);
  • [HSL] color: hsl(182, 20%, 50%);

To show opacity using the alpha value:

  • [RGBa] color: rgba(123, 20, 233, 0.5);
  • [HSLa] color: hsla(182, 20%, 50%, 0.5);

Background

  • -color
  • -image
  • -position

[Box Borders] Border-Style

border-style can have the following properties:

  • solid
  • dashed
  • dotted
  • double (two solid lines)
  • groove (carving border)
  • inset (border cut into the screen)
  • outset (border pop out of screen)
  • ridge (picture frame)
  • hidden or none

[Box Borders] Border-width

  • Use border-width property to set values
  • [Example] border-width: top right bottom left
  • [Example] border-width: 2px 5px 2px 5px

Example

Select a specific border by using the following property:

  • border-left-width
  • border-top-width
  • border-right-width
  • border-bottom-width

[Box Borders] - More

  • Use shorthanded property: "border: width style color" to specify all three properties in a single line
  • [Example] border: 5px double blue;

  • Border Example

  • Use "border-radius: size" to round off corners
  • [Example] border-radius: 20px;

  • Border Radius Example

CSS Content

  • padding: Adds invisible space within a box
  • margin: Adds invisible space outside a box
  • Both properties behave similar to the border property
  • display: inline, block, inline-block, and none
  • visibility visible, hidden
  • [display: none] will remove an element from view
  • [visibility: hidden] will show its reserved space

Box Model

  • [box-sizing] default to content-box
  • border-box is not affected by border thinkness or padding

Position

  • static (default)
  • relative
  • absolute
  • fixed: to a specific position to a page
  • [z-index] to change stack order of elements
  • [float] can be use to flush elements to the left or right
  • [clear] left, right, both, none (element will not touch designated side)
  • [line-height] to increase vertical spacing (only for text?)

Images

  • define dimensions using [width] and [height]
  • [background-image: url("")]
  • [background-repeat]
  • [background-position]
  • Shorthanded:[background: url("link") no-repeat right center;
  • [background-size] cover or contain
  • [background-attachment] default:scroll or fixed
  • (CSS3)[background-image: -webkit-linear-gradient(#666CCC, #BC1324);]
  • <img> when it communicates important info
  • [background] when used for styling page

Tables

  • <table>
  • <tr> table rows
  • <td> table data (columns)
  • <th> table heading [scope] "row" or "col"
  • [colspan] to span an element across more than 1 column
  • [rowspan] to span an element across more than 1 row
  • [thead]
  • [tbody]
  • [tfoot] used for sums
