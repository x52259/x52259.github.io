HTML and CSS Notes

Common HTML Elements

  • [Heading] Range from<h1> to <h6>
  • [Paragraph] Use <p> to start a paragraph
  • [breakLine] Use<br/>to start break up paragraph into a new line
  • [Ordered Lists] use <OL>
  • [Unordered Lists] use <UL>
  • [Link] Use<a href="link"></a>to enclose an element to make it clickable
  • [Image] Use<img src="insert link"/> to insert an image
  • [Alt] Use attributealt for the visually impaired by entering a description
  • [Comment-HTML] Use<!--Comment Here-->
  • [Comment-CSS] Use/* Comment Here */

CSS Setup

  • Use<style></style> in <head> to type css codes
  • Link stylesheet by using<link rel="stylesheet" type"text/css" href="name.css">

CSS Colors

There are four ways to indicate a color value:

  • [Name]color: Red;
  • [Hex]color: #FFFFFF;
  • [RGB]color: rgb(123, 20, 233);
  • [HSL]color: hsl(182, 20%, 50%);
  • [RGBa]color: rgba(123, 20, 233, 0.5);
  • [HSLa]color: hsla(182, 20%, 50%, 0.5);

Border

    border-style:can have the following properties:

  • solid;
  • dashed;
  • dotted;
  • double;(two solid lines)
  • groove;(carving border)
  • inset; (border cut into the screen)
  • outset;(border pop out of screen)
  • ridge;(picture frame)
  • hidden;ornone;
  • border-width: 2px 9px 2px 9px

  • Use border-radius: 20px;to round off corners

CSS Content

  • padding:Add invisible space within a box
  • margin:Add invisible space outside a box
  • Both properties behave similar to the border property
  • display:inline, block, inline-block, and none
  • visibilityvisible, hidden
  • display: none;will remove an element from view
  • visibility: hidden;will show its reserved space

Box Model

  • box-sizing: content-box;default value
  • box-sizing: border-box;size is not affected by border thinkness or padding

Position

  • static (default)
  • relative
  • absolute
  • fixed: to a specific position to a page
  • [z-index] to change stack order of elements
  • [float] can be use to flush elements to the left or right
  • [clear] left, right, both, none (element will not touch designated side)
  • [line-height] to increase vertical spacing (only for text?)

Images

  • background-repeat
  • background-position

  • background-size:cover; background-size: contain;
