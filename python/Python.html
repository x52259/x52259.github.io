  • [Comment] single-line #text
  • [Comment] multi-line """ text """
  • [valueType] type(insert_variable)
  • [raw_input] variable = raw_input("Insert question here")
  • [Index] python_index = "PYTHON"

  • [function] def func_name():

Comparators

  1. Equal to (==)
  2. Not equal to (!=)
  3. Less than (<)
  4. Less than or equal to (<=)
  5. Greater than (>)
  6. Greater than or equal to (>=)
  1. [not] returns True when, none is true
  2. [and] returns True when, both are true
  3. [or] returns True when, one is true
